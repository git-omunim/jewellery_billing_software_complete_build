Manual to Download the Drivers for oMunim Dongle

Please always open OMREVO drive from My Computer, Another CD Drive used only for Back-Up purpose of software.

 

 

 

Now Copy COM folder to your System C: drive.

 

 

 

Now Open COM folder.

Please check System Configuration, Check you have 32 Bit or 64 Bit OS.

 

If your System is 64-Bit then choose Run_X64.bat, Otherwise for 32-Bit system choose Run_X32.bat file.

Right Click on that file and click on Run as Administrator Option.

Click on YES Button in Next window

Then Successfully Message will populate.

If not able to install drive successfully, then try the same procedure from D drive. Copy the COM folder into D drive and run .BAT file.

Now start oMunim.exe from oMunim Dongle.

 

 

 

Now oMunim will start processing.

After processing login screen will display.
